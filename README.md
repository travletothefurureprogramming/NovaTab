# Nova Tab

### A modern and customizable browser start page designed to make every new tab useful.

NovaTab combines search, productivity tools, weather updates, quick links, and a clean interface into one dashboard.

---

## Features

- Modern dark design
- Real-time weather
- Command shortcuts:
  - /git → GitHub search
  - /ytb → YouTube search
  - /r → Reddit search
  - /chat → OpenAI chat shortcut
  - /mail → Gmail access
- Quick links support
- To-do system
- Calendar

---

## Tech Stack

- HTML, CSS, JavaScript
- Hosted with GitHub Pages
- Version control: Git & GitHub

---

## How to use

Visit:
https://travletothefurureprogramming.github.io/NovaTab/

- Use the search bar for normal Google searches (press Enter)
- Use shortcuts like `/ytb programming`
- Add and manage Quick Links (click to open, right-click to delete)
- Use the To-Do system (click to toggle, right-click to delete)
